# Device Preview - VS Code Extension

📱 **50+ cihazda web sitenizi önizleyin!** iPhone, Samsung, Pixel, Xiaomi, iPad ve daha fazlası.

## Özellikler

- ✅ 50+ mobil cihaz ve tablet
- ✅ Tıkla-büyüt ile detaylı inceleme
- ✅ iPhone, Samsung, Pixel, Xiaomi, OnePlus, Huawei, Oppo
- ✅ iPad, Galaxy Tab, iPad Pro
- ✅ Portrait / Landscape döndürme
- ✅ Filtreleme (iPhone / Samsung / Android / Tablet)
- ✅ Önceki / Sonraki ile gezinme
- ✅ ESC ile kapatma
- ✅ Klavye kısayolu (Ctrl+Alt+D)
- ✅ Tamamen ücretsiz ve açık kaynak

## Kullanım

1. `Ctrl+Shift+P` → "Device Preview: Cihaz Önizleme" yazın
2. Veya `Ctrl+Alt+D` kısayolunu kullanın
3. URL girin (örn: `http://localhost:5173`)
4. Cihazlara tıklayın, büyütün, inceleyin

## Kısayollar

| Tuş | İşlem |
|-----|-------|
| `Ctrl+Alt+D` | Cihaz Önizleme aç |
| `← →` | Önceki / Sonraki cihaz |
| `ESC` | Kapat |
| Tıkla | Cihazı büyüt |
